# Enumeration: LastPriceAnimationMode

Represents the type of the last price animation for series such as area
or line.

## Enumeration Members[​](LastPriceAnimationMode.html#enumeration-members "Direct link to Enumeration Members"){.hash-link aria-label="Direct link to Enumeration Members"} {#enumeration-members .anchor .anchorWithStickyNavbar_LWe7}

### Disabled[​](LastPriceAnimationMode.html#disabled "Direct link to Disabled"){.hash-link aria-label="Direct link to Disabled"} {#disabled .anchor .anchorWithStickyNavbar_LWe7}

> **Disabled**: `0`

Animation is always disabled

------------------------------------------------------------------------

### Continuous[​](LastPriceAnimationMode.html#continuous "Direct link to Continuous"){.hash-link aria-label="Direct link to Continuous"} {#continuous .anchor .anchorWithStickyNavbar_LWe7}

> **Continuous**: `1`

Animation is always enabled.

------------------------------------------------------------------------

### OnDataUpdate[​](LastPriceAnimationMode.html#ondataupdate "Direct link to OnDataUpdate"){.hash-link aria-label="Direct link to OnDataUpdate"} {#ondataupdate .anchor .anchorWithStickyNavbar_LWe7}

> **OnDataUpdate**: `2`

Animation is active after new data.
