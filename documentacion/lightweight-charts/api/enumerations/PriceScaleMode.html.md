# Enumeration: PriceScaleMode

Represents the price scale mode.

## Enumeration Members[​](PriceScaleMode.html#enumeration-members "Direct link to Enumeration Members"){.hash-link aria-label="Direct link to Enumeration Members"} {#enumeration-members .anchor .anchorWithStickyNavbar_LWe7}

### Normal[​](PriceScaleMode.html#normal "Direct link to Normal"){.hash-link aria-label="Direct link to Normal"} {#normal .anchor .anchorWithStickyNavbar_LWe7}

> **Normal**: `0`

Price scale shows prices. Price range changes linearly.

------------------------------------------------------------------------

### Logarithmic[​](PriceScaleMode.html#logarithmic "Direct link to Logarithmic"){.hash-link aria-label="Direct link to Logarithmic"} {#logarithmic .anchor .anchorWithStickyNavbar_LWe7}

> **Logarithmic**: `1`

Price scale shows prices. Price range changes logarithmically.

------------------------------------------------------------------------

### Percentage[​](PriceScaleMode.html#percentage "Direct link to Percentage"){.hash-link aria-label="Direct link to Percentage"} {#percentage .anchor .anchorWithStickyNavbar_LWe7}

> **Percentage**: `2`

Price scale shows percentage values according the first visible value of
the price scale. The first visible value is 0% in this mode.

------------------------------------------------------------------------

### IndexedTo100[​](PriceScaleMode.html#indexedto100 "Direct link to IndexedTo100"){.hash-link aria-label="Direct link to IndexedTo100"} {#indexedto100 .anchor .anchorWithStickyNavbar_LWe7}

> **IndexedTo100**: `3`

The same as percentage mode, but the first value is moved to 100.
