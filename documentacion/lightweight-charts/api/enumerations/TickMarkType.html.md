# Enumeration: TickMarkType

Represents the type of a tick mark on the time axis.

## Enumeration Members[​](TickMarkType.html#enumeration-members "Direct link to Enumeration Members"){.hash-link aria-label="Direct link to Enumeration Members"} {#enumeration-members .anchor .anchorWithStickyNavbar_LWe7}

### Year[​](TickMarkType.html#year "Direct link to Year"){.hash-link aria-label="Direct link to Year"} {#year .anchor .anchorWithStickyNavbar_LWe7}

> **Year**: `0`

The start of the year (e.g. it\'s the first tick mark in a year).

------------------------------------------------------------------------

### Month[​](TickMarkType.html#month "Direct link to Month"){.hash-link aria-label="Direct link to Month"} {#month .anchor .anchorWithStickyNavbar_LWe7}

> **Month**: `1`

The start of the month (e.g. it\'s the first tick mark in a month).

------------------------------------------------------------------------

### DayOfMonth[​](TickMarkType.html#dayofmonth "Direct link to DayOfMonth"){.hash-link aria-label="Direct link to DayOfMonth"} {#dayofmonth .anchor .anchorWithStickyNavbar_LWe7}

> **DayOfMonth**: `2`

A day of the month.

------------------------------------------------------------------------

### Time[​](TickMarkType.html#time "Direct link to Time"){.hash-link aria-label="Direct link to Time"} {#time .anchor .anchorWithStickyNavbar_LWe7}

> **Time**: `3`

A time without seconds.

------------------------------------------------------------------------

### TimeWithSeconds[​](TickMarkType.html#timewithseconds "Direct link to TimeWithSeconds"){.hash-link aria-label="Direct link to TimeWithSeconds"} {#timewithseconds .anchor .anchorWithStickyNavbar_LWe7}

> **TimeWithSeconds**: `4`

A time with seconds.
