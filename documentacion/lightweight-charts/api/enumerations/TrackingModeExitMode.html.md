# Enumeration: TrackingModeExitMode

Determine how to exit the tracking mode.

By default, mobile users will long press to deactivate the scroll and
have the ability to check values and dates. Another press is required to
activate the scroll, be able to move left/right, zoom, etc.

## Enumeration Members[​](TrackingModeExitMode.html#enumeration-members "Direct link to Enumeration Members"){.hash-link aria-label="Direct link to Enumeration Members"} {#enumeration-members .anchor .anchorWithStickyNavbar_LWe7}

### OnTouchEnd[​](TrackingModeExitMode.html#ontouchend "Direct link to OnTouchEnd"){.hash-link aria-label="Direct link to OnTouchEnd"} {#ontouchend .anchor .anchorWithStickyNavbar_LWe7}

> **OnTouchEnd**: `0`

Tracking Mode will be deactivated on touch end event.

------------------------------------------------------------------------

### OnNextTap[​](TrackingModeExitMode.html#onnexttap "Direct link to OnNextTap"){.hash-link aria-label="Direct link to OnNextTap"} {#onnexttap .anchor .anchorWithStickyNavbar_LWe7}

> **OnNextTap**: `1`

Tracking Mode will be deactivated on the next tap event.
