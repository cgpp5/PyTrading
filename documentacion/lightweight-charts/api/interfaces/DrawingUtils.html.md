# Interface: DrawingUtils

Helper drawing utilities exposed by the library to a Primitive (a.k.a
plugin).

### setLineStyle()[​](DrawingUtils.html#setlinestyle "Direct link to setLineStyle()"){.hash-link aria-label="Direct link to setLineStyle()"} {#setlinestyle .anchor .anchorWithStickyNavbar_LWe7}

> `readonly` **setLineStyle**: (`ctx`, `lineStyle`) =\> `void`

Drawing utility to change the line style on the canvas context to one of
the built-in line styles.

#### Parameters[​](DrawingUtils.html#parameters "Direct link to Parameters"){.hash-link aria-label="Direct link to Parameters"} {#parameters .anchor .anchorWithStickyNavbar_LWe7}

• **ctx**: `CanvasRenderingContext2D`

2D rendering context for the target canvas.

• **lineStyle**: [`LineStyle`](../enumerations/LineStyle.html)

Built-in [LineStyle](../enumerations/LineStyle.html) to set on the
canvas context.
