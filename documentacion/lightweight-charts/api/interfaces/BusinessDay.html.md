# Interface: BusinessDay

Represents a time as a day/month/year.

## Example[​](BusinessDay.html#example "Direct link to Example"){.hash-link aria-label="Direct link to Example"} {#example .anchor .anchorWithStickyNavbar_LWe7}

``` {.prism-code .language-js .codeBlock_bY9V .thin-scrollbar tabindex="0" style="color:#393A34;background-color:#f6f8fa"}
const day = { year: 2019, month: 6, day: 1 }; // June 1, 2019
```
