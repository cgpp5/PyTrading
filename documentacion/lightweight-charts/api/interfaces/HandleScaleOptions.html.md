# Interface: HandleScaleOptions

Represents options for how the chart is scaled by the mouse and touch
gestures.

## Properties[​](HandleScaleOptions.html#properties "Direct link to Properties"){.hash-link aria-label="Direct link to Properties"} {#properties .anchor .anchorWithStickyNavbar_LWe7}

### mouseWheel[​](HandleScaleOptions.html#mousewheel "Direct link to mouseWheel"){.hash-link aria-label="Direct link to mouseWheel"} {#mousewheel .anchor .anchorWithStickyNavbar_LWe7}

> **mouseWheel**: `boolean`

Enable scaling with the mouse wheel.

#### Default Value[​](HandleScaleOptions.html#default-value "Direct link to Default Value"){.hash-link aria-label="Direct link to Default Value"} {#default-value .anchor .anchorWithStickyNavbar_LWe7}

`true`

------------------------------------------------------------------------

### pinch[​](HandleScaleOptions.html#pinch "Direct link to pinch"){.hash-link aria-label="Direct link to pinch"} {#pinch .anchor .anchorWithStickyNavbar_LWe7}

> **pinch**: `boolean`

Enable scaling with pinch/zoom gestures.

#### Default Value[​](HandleScaleOptions.html#default-value-1 "Direct link to Default Value"){.hash-link aria-label="Direct link to Default Value"} {#default-value-1 .anchor .anchorWithStickyNavbar_LWe7}

`true`

------------------------------------------------------------------------

### axisPressedMouseMove[​](HandleScaleOptions.html#axispressedmousemove "Direct link to axisPressedMouseMove"){.hash-link aria-label="Direct link to axisPressedMouseMove"} {#axispressedmousemove .anchor .anchorWithStickyNavbar_LWe7}

> **axisPressedMouseMove**: `boolean` \|
> [`AxisPressedMouseMoveOptions`](AxisPressedMouseMoveOptions.html)

Enable scaling the price and/or time scales by holding down the left
mouse button and moving the mouse.

------------------------------------------------------------------------

### axisDoubleClickReset[​](HandleScaleOptions.html#axisdoubleclickreset "Direct link to axisDoubleClickReset"){.hash-link aria-label="Direct link to axisDoubleClickReset"} {#axisdoubleclickreset .anchor .anchorWithStickyNavbar_LWe7}

> **axisDoubleClickReset**: `boolean` \|
> [`AxisDoubleClickOptions`](AxisDoubleClickOptions.html)

Enable resetting scaling by double-clicking the left mouse button.
